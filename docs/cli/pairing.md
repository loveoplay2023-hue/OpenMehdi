# `openmehdi pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
openmehdi pairing list whatsapp
openmehdi pairing approve whatsapp <code> --notify
```
